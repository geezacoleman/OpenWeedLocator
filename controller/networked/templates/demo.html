OpenWeedLocator

VegOWL

Open-source weed detection built on Raspberry Pi. Each camera identifies weeds in real time and triggers spot spraying, reducing herbicide use. Working with the Warren Cauliflower Group for precision application in brassica crops.

Using hybrid detection: a neural network identifies the crop, then colour-based algorithms detect everything else that's green as a weed. Better field data means better models — help us improve by contributing imagery from your region.
